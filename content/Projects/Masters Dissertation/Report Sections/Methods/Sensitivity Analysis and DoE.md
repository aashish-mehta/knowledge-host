# Sensitivity Analysis and DoE

Performed sensitivity testing by doubling each geometric parameter from a control:

- Variables: β1, β2, α2, L1, L2
- Measured pressure head change
- Informed refinement bounds for [[Design of Experiments]]
- [[D-optimal]] was used to define the DoE
- Generated in MATLAB and exported for use in the simulation framework
